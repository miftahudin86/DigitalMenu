# 🍽️ DigitalMenu

DigitalMenu adalah aplikasi berbasis **CodeIgniter 4** yang dirancang untuk mempermudah pengelolaan menu makanan dan minuman secara digital. Aplikasi ini menyediakan dashboard admin untuk mengelola kategori, menu, dan pesanan pelanggan serta dapat dijalankan menggunakan **Docker** sehingga proses deployment menjadi lebih mudah dan konsisten.

---

## ✨ Fitur

- 📋 Manajemen Menu
- 📂 Manajemen Kategori
- 📊 Dashboard Admin
- 🛒 Pemesanan Menu
- 🖼 Upload Gambar Menu
- 👥 Halaman Customer
- 📱 Tampilan Responsif
- 🐳 Deployment menggunakan Docker

---

## 🛠️ Teknologi

- CodeIgniter 4
- PHP 8
- MySQL 8
- Docker
- HTML5
- CSS3
- Bootstrap 5

---

## 🚀 Cara Menjalankan

### Clone Repository

```bash
git clone https://github.com/miftahudin86/DigitalMenu.git
```

### Masuk ke Folder Project

```bash
cd DigitalMenu
```

### Jalankan Docker

```bash
docker compose up -d --build
```

### Buka Browser

```
http://localhost:8080
```
